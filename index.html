True compatibility

Emulation projects are a massive win, but they aren't fully compatible with everything yet. The most common emulator currently supports around 87% of Flash, but it isn't enough.

FlashForCurrent gets past this by using the actual Pepper Flash engine. This means that everything will work exactly as it did just before flash went EOL.

Notice: Due to the way this works, only one Flash tab can be loaded at a time. Loading a new Flash tab will cause the previous tabs to unload.
